🌤 Weather App (Vanilla JavaScript)

A simple weather application built using HTML, CSS, and vanilla JavaScript.

🔹 Features

Search weather by city name

Fetch real-time weather data from a public API

Display temperature and weather conditions

Dynamic DOM updates based on user input

Responsive layout

🛠 Technologies Used

HTML5

CSS3

JavaScript (ES6)

Fetch API

🚀 How to Run

Clone the repository

Open index.html in your browser
